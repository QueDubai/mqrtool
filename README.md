## 本地开发

```bash
npm install
npm run dev
```

浏览器打开 `http://localhost:5173`。

## 构建

```bash
npm run build
npm run preview
```

## 部署到 GitHub Pages

1. 推送到 `QueDubai/mqrtool` 的 `main` 分支
2. 仓库 **Settings → Pages → Build and deployment** 选择 **GitHub Actions**
3. 推送后等待 Actions 完成

访问地址：

```text
https://quedubai.github.io/mqrtool/
```

## 历史记录

最近 10 次生成记录保存在浏览器 `localStorage`，键名 `sto-qrcode-history`。
